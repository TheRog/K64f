The FNET is Dual TCP/IPv4/IPv6 stack (under GNU LGPLv3 license) for building embedded communication software on 32bit MCUs.

The stack provides following protocols and services:
  • Supported Platforms:
    • ARM Cortex-M4, Kinetis:
      • MK64FN1M (FRDM-K64F)
      • MK60N512 (TWR-K60N512)
      • MK70FN1M (TWR-K70F120M)
      • MK60FN1M (TWR-K60F120M)
    • ColdFire:
      • Modelo - MCF54418 (TWR-MCF5441x)
      • Kirin2 - MCF52235 (M52235EVB)
      • Kirin3 - MCF52259 (M52259EVB, M52259DEMO, TWR-MCF52259)
    • Qorivva Power Architecture:
      • Fado - MPC5668G (MPC5668GKIT)
  • Supported Compilers:
    • ARM:
      • Keil uVisionv v4.7
    • IAR:
      • Embedded Workbench for ARM v7.1
    • Freescale:
      • Kinetis Design Studio v3.0
      • CodeWarrior Development Studio for Microcontrollers v10.6
      • CodeWarrior for ColdFire v7.2.2
      • CodeWarrior Development Studio for MPC55xx/MPC56xx v2.10
  • Bare-metal TCP/IP stack.
  • Non-blocking Socket layer API (STREAM, DGRAM, RAW).
  • Core Protocols:
    • TCP
    • UDP
    • IPv4, ICMPv4, IGMP, ARP
    • IPv6, ICMPv6, MLD, ND
  • Network Interfaces:
    • Ethernet
    • Loopback
  • Services:
    • HTTP 1.0/0.9 Server:
      • GET, POST requests
      • CGI, SSI
      • Basic Access Authentication
    • TELNET Server
    • DNS Client
    • LLMNR Server
    • DHCP Client
    • TFTP Server/Client
    • PING Service
    • ROM File System
    • Flash Memory Driver
    • Command Line Shell
    • Serial Driver
  • Tools:
    • GUI File System Generation Tool
    • GUI Throughput Benchmark Tool
  • Applications:
    • TFTP Bootloader
    • Shell Application
    • Throughput Benchmark Application
